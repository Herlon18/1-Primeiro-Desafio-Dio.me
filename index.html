    Links

    Tag < a href="" target=""> < /a > utilizada para links de referências, seja dentro da mesma pagina, levar para outra pagína dentro da mesma aba, ou um link externo onde é aberto uma outra página em outra aba do navegador.
    Essa tag funciona em conjunto com seu atributo href="", dentro da 1ª aspas do atributo href="" é colocado o link para sua refêrencia, na 2ª aspas do atributo target é colocado como você deseja abrir este link, se na mesma pagina ou em uma nova aba, caso não seja colocado esse atributo, por padrão o link colocado no href é aberto na mesma aba.
    Entre a Tag é Digitado o Texto que irá se transformar em um Link Clicável
